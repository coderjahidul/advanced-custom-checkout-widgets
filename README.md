## ğŸ“– How To Use

1.  Edit any page using **Elementor**.
2.  Search for the **Advanced Checkout** widget in the widget panel.
3.  Drag and drop it onto your page.
4.  **Content Settings**:
    *   **Select Products to Show**: Enter comma-separated Product IDs to display a selection grid at the top of the checkout.
    *   **Pre-selected Products**: Enter comma-separated Product IDs to have them automatically added to the cart when a user visits the page.
5.  **Style Settings**: Go to the **Style** tab to customize colors, buttons, and form indicators to match your brand's theme.
