Build an advanced custom checkout plugin for WooCommerce that works with Elementor. Support single/multiple product selection, AJAX add-to-cart system, and modern UI.

== Description ==

Elevate your WooCommerce checkout experience with the **Advanced Custom Checkout Widgets**. This plugin provides a highly customizable Elementor widget that transforms your standard WooCommerce checkout into a modern, conversion-optimized, single-page experience.

= âœ¨ Key Features =

* **Modern Checkout Template**: A sleek, two-column layout designed for maximum conversions.
* **Product Selection Grid**: Display a curated selection of products directly on the checkout page. Users can add products with one click without leaving the page.
* **Pre-selected Products**: Automatically add specific products to the cart when the checkout page is loaded.
* **AJAX Add-to-Cart**: Products are added to the cart instantly via AJAX, with automatic order review and total updates.
* **Sticky Order Review**: Keeps the order summary and payment methods visible while the user fills out the billing and shipping forms.
* **Elementor Native Controls**: Full control over designs directly from the Elementor editor.

== Installation ==

1. Upload the `advanced-custom-checkout-widgets` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Ensure WooCommerce and Elementor are already active.

== Frequently Asked Questions ==

= Does it work with Elementor Free? =
Yes, it works with both Elementor Free and Pro versions.

= How do I add the checkout to my page? =
Simply edit your page with Elementor, search for the "Advanced Checkout" widget, and drag it into your layout.
